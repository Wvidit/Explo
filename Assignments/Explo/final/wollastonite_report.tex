\begin{abstract}
We present a six-phase computational materials design pipeline for porous Wollastonite (CaSiO\textsubscript{3}) ceramics that couples stochastic microstructure generation with multi-physics homogenisation and gradient boosting regression surrogates. Three-dimensional representative volume elements (RVEs) are generated via Voronoi tessellation with preferential triple-junction porosity injection. Effective thermal, elastic, and thermo-mechanical properties are computed through either finite element (FE) homogenisation using SOLID226 coupled-field elements or a novel set of \emph{microstructure-aware analytical bounds} that augment classical Maxwell--Eucken, Mori--Tanaka, and Turner rule-of-mixtures models with grain-boundary scattering, pore-morphology tortuosity, and Hall--Petch-type corrections parameterised by chord-length descriptors. The analytical fallback incorporates controlled stochastic perturbation ($\text{CV} \approx 1.5$--$2.0\%$) to emulate finite element scatter, enabling the ML surrogate to learn non-trivial, multi-dimensional structure--property mappings across a 10-feature microstructural descriptor space. Trained on 200 stochastic configurations, the gradient boosting regressors achieve $R^2 = 0.80$ (thermal conductivity), $0.87$ (CTE), $0.95$ (density), and $0.85$ (Vickers hardness), with a database-fitted polynomial inverse design module for target property optimisation.
\end{abstract}

\section{Introduction}
\label{sec:intro}

Wollastonite (CaSiO\textsubscript{3}), a chain-silicate mineral of the pyroxenoid group, exhibits a monoclinic crystal structure ($P\bar{1}$ space group) with characteristic needle-like morphology along the $b$-axis. Its combination of low thermal conductivity ($k \approx 2.6$--$3.5$~\si{W.m^{-1}.K^{-1}}$), moderate elastic stiffness ($E \approx 93$--$96$~\si{GPa}$), and thermal stability up to $\sim$\SI{1000}{\celsius} makes it a candidate for thermal barrier coatings, friction composites, and bioactive ceramic scaffolds~\cite{pirkelmann2023}.

In sintered ceramic bodies, macroscopic thermomechanical performance is governed by the interplay between intrinsic single-crystal properties and the statistical descriptors of the porous microstructure: volume fraction of porosity $\phi$, pore morphology (aspect ratio, tortuosity), grain size distribution, and grain-boundary density. Full-field finite element homogenisation on digitised microstructures remains the gold standard for capturing these couplings, but requires commercial solver licenses (e.g., ANSYS Mechanical) and substantial computational cost ($\sim$10--60~min per RVE at $\sim$50k DOF).

This work introduces a \emph{dual-mode} pipeline that:
\begin{enumerate}
    \item Generates stochastic 3D Voronoi RVEs with controllable porosity and grain count (Phase~2);
    \item Performs coupled thermal--structural FE homogenisation when an ANSYS/PyMAPDL license is detected, or automatically falls back to microstructure-aware analytical bounds when it is not (Phases~3--4);
    \item Extracts six engineering properties augmented with grain-size and pore-morphology corrections (Phase~5);
    \item Trains gradient boosting regressors on a 10-feature descriptor space and provides a polynomial-fitted inverse design module (Phase~6).
\end{enumerate}

\section{Phase 2: Stochastic RVE Microstructure Generation}
\label{sec:phase2}

\subsection{Voronoi Tessellation}

The RVE domain $\Omega = [0, N]^3$ (voxel resolution $N = 64$) is tessellated into $N_s$ Voronoi cells from $N_s$ seed points drawn from a uniform random distribution $\mathbf{x}_i \sim \mathcal{U}([0, N]^3)$, $i = 1, \ldots, N_s$. Each voxel $\mathbf{v}$ is assigned to the nearest seed:
\begin{equation}
    g(\mathbf{v}) = \arg\min_{i \in \{1,\ldots,N_s\}} \|\mathbf{v} - \mathbf{x}_i\|_2
\end{equation}
implemented via a $k$-d tree query (\texttt{scipy.spatial.cKDTree}) with $\mathcal{O}(N^3 \log N_s)$ complexity.

\subsection{Triple-Junction Porosity Injection}

Porosity is localised at grain triple-junctions to emulate imperfect sintering densification. A $3 \times 3 \times 3$ uniform filter computes the local grain multiplicity:
\begin{equation}
    m(\mathbf{v}) = |\{g(\mathbf{u}) : \mathbf{u} \in \mathcal{N}_{27}(\mathbf{v})\}|
\end{equation}
Voxels satisfying $m(\mathbf{v}) \geq 3$ are candidates for pore assignment. From this candidate set $\mathcal{C}$, exactly $n_p = \lfloor \phi_{\text{target}} \cdot N^3 \rfloor$ voxels are randomly selected without replacement and set to the pore phase ($\text{RVE}(\mathbf{v}) = 1$).

\subsection{Microstructural Descriptors}
\label{sec:descriptors}

Chord-length analysis is performed independently along each Cartesian axis $d \in \{x, y, z\}$. For axis $d$, rays are cast through all $N^2$ voxel lines. Each contiguous run of matrix voxels ($\text{RVE} = 0$) constitutes a chord of length $\ell$. The first two moments define the descriptor set:
\begin{equation}
    \bar{c}_d = \mathbb{E}[\ell_d], \qquad \sigma^2_{c,d} = \text{Var}[\ell_d]
\end{equation}

The full feature vector for ML training comprises 10 descriptors:
\begin{equation}
    \mathbf{x} = \left(\phi,\; \bar{c}_x,\; \bar{c}_y,\; \bar{c}_z,\; \sigma^2_{c,x},\; \sigma^2_{c,y},\; \sigma^2_{c,z},\; \bar{c}_{\text{avg}},\; N_s,\; N\right)^{\top}
    \label{eq:features}
\end{equation}
where $\bar{c}_{\text{avg}} = (\bar{c}_x + \bar{c}_y + \bar{c}_z)/3$.

\section{Phase 3: Geometry Construction and Meshing}
\label{sec:phase3}

The macroscopic test specimen is a cylinder of radius $R = 4$~\si{mm} and height $H = 4$~\si{mm}. In FE mode, the domain is discretised with SOLID226 coupled thermal--structural hexahedral elements (KEYOPT(1)=11 for thermo-structural coupling) at an element edge length of $\sim$\SI{0.5}{mm}, yielding approximately 12,000 elements and 14,000 nodes.

Phase mapping assigns material number 1 (Wollastonite) or 2 (air) to each element based on the nearest RVE voxel value, using a $k$-d tree mapping from element centroids to the $64^3$ voxel grid.

In analytical fallback mode, a lightweight mock object replaces the MAPDL session, preserving the API contract for downstream phases while carrying the RVE array and computed porosity.

\section{Phase 4: Multi-Physics Homogenisation}
\label{sec:phase4}

\subsection{FE Homogenisation (PyMAPDL)}

Three boundary value problems (BVPs) are solved on the discretised cylinder:

\paragraph{Thermal conductivity.}
Steady-state heat conduction with Dirichlet conditions $T|_{z=0} = T_{\text{cold}}$, $T|_{z=H} = T_{\text{hot}} = T_{\text{cold}} + \Delta T$ and adiabatic radial surfaces. Fourier's law yields:
\begin{equation}
    k_{\text{eff}} = \frac{Q_{\text{total}} \cdot H}{A \cdot \Delta T}
    \label{eq:fourier}
\end{equation}
where $Q_{\text{total}} = \sum_{i \in \Gamma_{\text{hot}}} q_i \cdot A_i$ is the total heat flux through the hot face and $A = \pi R^2$.

\paragraph{Elastic modulus.}
A uniaxial strain BVP imposes $\varepsilon_{zz} = -10^{-3}$ on the top face with full fixity ($u_x = u_y = u_z = 0$) on the bottom. The effective Young's modulus under the Voigt--Reuss assumption of macroscopic isotropy is:
\begin{equation}
    E_{\text{eff}} = \frac{\langle \sigma_{zz} \rangle}{\varepsilon_{zz}} = \frac{\sum_{i \in \Gamma_{\text{top}}} F_{z,i}}{A \cdot \varepsilon_{zz}}
    \label{eq:youngs}
\end{equation}

\paragraph{Coefficient of thermal expansion.}
A uniform temperature increment $\Delta T = 400$~\si{K} is applied with minimal rigid-body constraints (3-2-1 scheme). The volumetric strain is computed from nodal displacements on the outer surfaces, yielding:
\begin{equation}
    \alpha_{\text{eff}} = \frac{1}{3} \cdot \frac{\Delta V / V_0}{\Delta T}
    \label{eq:cte}
\end{equation}

\subsection{Microstructure-Aware Analytical Fallback}
\label{sec:fallback}

When the ANSYS solver is unavailable, three classical homogenisation models are augmented with physically motivated correction factors derived from the chord-length descriptors (\S\ref{sec:descriptors}).

\subsubsection{Thermal Conductivity: Augmented Maxwell--Eucken}

The Maxwell--Eucken upper bound for a matrix--inclusion composite with spherical pores gives:
\begin{equation}
    k_{\text{ME}} = k_m \cdot \frac{k_m + 2k_p + 2\phi(k_p - k_m)}{k_m + 2k_p - \phi(k_p - k_m)}
    \label{eq:maxwell}
\end{equation}

This base model is corrected by four multiplicative factors:
\begin{equation}
    k_{\text{eff}} = k_{\text{ME}} \cdot f_{\text{gb}} \cdot f_{\tau} \cdot f_{\sigma^2} \cdot f_{N_s} \cdot (1 + \eta_k)
    \label{eq:k_corrected}
\end{equation}
where:

\paragraph{Grain-boundary scattering ($f_{\text{gb}}$).}
Phonon scattering at grain boundaries is modelled as inversely proportional to the normalised mean chord length $\tilde{c} = \bar{c}_{\text{avg}} / N$:
\begin{equation}
    f_{\text{gb}} = 1 - \alpha_{\text{gb}}(1 - \tilde{c}), \qquad \alpha_{\text{gb}} = 0.25
\end{equation}
This captures the Kapitza resistance at grain boundaries: finer grains ($\tilde{c} \to 0$) reduce $k_{\text{eff}}$ by up to $25\%$ relative to a single crystal.

\paragraph{Tortuosity correction ($f_\tau$).}
Chord-length anisotropy $\mathcal{A} = (\bar{c}_{\max} - \bar{c}_{\min})/\bar{c}_{\max}$ serves as a proxy for pore-channel tortuosity:
\begin{equation}
    f_\tau = 1 - \beta_\tau \cdot \mathcal{A}, \qquad \beta_\tau = 0.20
\end{equation}

\paragraph{Pore irregularity ($f_{\sigma^2}$).}
The mean chord-length variance $\bar{\sigma}^2_c = (\sigma^2_{c,x} + \sigma^2_{c,y} + \sigma^2_{c,z})/3$ quantifies pore shape heterogeneity:
\begin{equation}
    f_{\sigma^2} = 1 - \gamma_{\sigma^2} \cdot \bar{\sigma}^2_c, \qquad \gamma_{\sigma^2} = 1.5 \times 10^{-4}
\end{equation}

\paragraph{Grain count correction ($f_{N_s}$).}
At fixed porosity, increasing the number of Voronoi seeds produces a finer grain structure with more boundary surface area per unit volume:
\begin{equation}
    f_{N_s} = 1 - \delta_{N_s}(N_s - 50), \qquad \delta_{N_s} = 8 \times 10^{-4}
\end{equation}

\paragraph{Stochastic noise ($\eta_k$).}
A sample-specific Gaussian perturbation $\eta_k \sim \mathcal{N}(0, 0.02^2)$ with a deterministic seed derived from the MD5 hash of the RVE voxel array ensures reproducibility while emulating finite element numerical scatter.

Additionally, the reference temperature is sampled as $T_{\text{eval}} = T_{\text{ref}} + \mathcal{U}(-30, 30)$ to introduce natural property-temperature coupling scatter.

\subsubsection{Elastic Modulus: Augmented Mori--Tanaka}

The Mori--Tanaka estimate for a matrix containing spherical voids ($E_p = 0$, $\nu_p = 0$) reduces to:
\begin{equation}
    E_{\text{MT}} = E_m \cdot \frac{1 - \phi}{1 + \phi \cdot \frac{1 - \nu_m}{2(1 - 2\nu_m)}}
    \label{eq:moritanaka}
\end{equation}

The corrected effective modulus introduces pore-morphology and grain-size effects:
\begin{equation}
    E_{\text{eff}} = E_{\text{MT}} \cdot f_{\text{asp}} \cdot f_{\text{grain}} \cdot f_{N_s}^{(E)} \cdot f_{\mathcal{A}}^{(E)} \cdot (1 + \eta_E)
\end{equation}

\paragraph{Pore aspect ratio ($f_{\text{asp}}$).}
The coefficient of variation of inter-axis chord variances $\text{CV}_\sigma = \text{std}(\sigma^2_{c,x}, \sigma^2_{c,y}, \sigma^2_{c,z}) / \text{mean}(\cdot)$ serves as a morphological anisotropy measure:
\begin{equation}
    f_{\text{asp}} = 1 - \alpha_{\text{asp}} \cdot \text{CV}_\sigma, \qquad \alpha_{\text{asp}} = 0.40
\end{equation}
High $\text{CV}_\sigma$ indicates elongated, non-spherical pores --- departing from the spherical-pore assumption of Eq.~\eqref{eq:moritanaka} --- and reduces stiffness.

\paragraph{Grain-size correction ($f_{\text{grain}}$).}
A bidirectional correction centred at $\tilde{c} = 0.5$ prevents systematic bias:
\begin{equation}
    f_{\text{grain}} = 1 + \beta_{\text{grain}}(0.5 - \tilde{c}), \qquad \beta_{\text{grain}} = 0.12
\end{equation}

\paragraph{Grain-count and anisotropy corrections.}
\begin{align}
    f_{N_s}^{(E)} &= 1 + \gamma_{N_s}^{(E)} (N_s - 85), &\quad \gamma_{N_s}^{(E)} &= 4 \times 10^{-4} \\
    f_{\mathcal{A}}^{(E)} &= 1 - \delta_{\mathcal{A}}^{(E)} \cdot \frac{\text{std}(\bar{c}_x, \bar{c}_y, \bar{c}_z)}{\text{mean}(\bar{c}_x, \bar{c}_y, \bar{c}_z)}, &\quad \delta_{\mathcal{A}}^{(E)} &= 0.20
\end{align}

\subsubsection{CTE: Augmented Turner Rule of Mixtures}

The Turner (Voigt) rule of mixtures for effective CTE:
\begin{equation}
    \alpha_{\text{ROM}} = \alpha_m(1 - \phi) + \alpha_p \phi
    \label{eq:turner}
\end{equation}

is corrected for grain-boundary constraint and microstructural anisotropy:
\begin{equation}
    \alpha_{\text{eff}} = \alpha_{\text{ROM}} \cdot f_{\text{gb}}^{(\alpha)} \cdot f_{\text{size}}^{(\alpha)} \cdot f_{\mathcal{A}}^{(\alpha)} \cdot f_{\sigma^2}^{(\alpha)} \cdot (1 + \eta_\alpha)
\end{equation}

\begin{align}
    f_{\text{gb}}^{(\alpha)}  &= 1 - \alpha_{\text{gb}}^{(\alpha)}(N_s - 50),  &\quad \alpha_{\text{gb}}^{(\alpha)}  &= 1.0 \times 10^{-3} \\
    f_{\text{size}}^{(\alpha)} &= 1 - \beta_{\text{size}}^{(\alpha)}(1 - \tilde{c}), &\quad \beta_{\text{size}}^{(\alpha)} &= 0.10 \\
    f_{\mathcal{A}}^{(\alpha)} &= 1 + \gamma_{\mathcal{A}}^{(\alpha)} \cdot \text{CV}_{\bar{c}}, &\quad \gamma_{\mathcal{A}}^{(\alpha)} &= 0.12 \\
    f_{\sigma^2}^{(\alpha)}    &= 1 - \delta_{\sigma^2}^{(\alpha)} \cdot \bar{\sigma}^2_c, &\quad \delta_{\sigma^2}^{(\alpha)} &= 8.0 \times 10^{-5}
\end{align}

\section{Phase 5: Property Extraction with Empirical Corrections}
\label{sec:phase5}

\subsection{Density and Porosity}

Effective density follows the rule of mixtures with a grain-boundary deficit correction:
\begin{equation}
    \rho_{\text{eff}} = (1 - \phi)\rho_m + \phi\rho_p - \rho_m \cdot \xi_{\text{gb}}(1 - \tilde{c})
\end{equation}
where $\xi_{\text{gb}} = 0.003$ accounts for the slightly lower packing density at grain-boundary regions. A measurement noise term $\eta_\rho \sim \mathcal{N}(0, 0.0015^2)$ is applied multiplicatively.

\subsection{Vickers Hardness: Rice--Duckworth Model with Hall--Petch Correction}

The Rice--Duckworth exponential porosity model for ceramic hardness~\cite{rice1998} is augmented with a stiffness-ratio scaling term and three microstructural corrections:
\begin{equation}
    HV = HV_{\text{dense}} \cdot \exp(-b\phi) \cdot \frac{E_{\text{eff}}}{E_{\text{dense}}} \cdot f_{\text{HP}} \cdot f_{\text{var}}^{(HV)} \cdot f_{N_s}^{(HV)} \cdot (1 + \eta_{HV})
    \label{eq:hardness}
\end{equation}
where $HV_{\text{dense}} = 570$~HV, $b = 7.0$, and $E_{\text{dense}} = E_m(0\text{\textdegree C}) = 96$~\si{GPa}.

\paragraph{Hall--Petch-type grain-size correction.}
Following the classical Hall--Petch relationship $\sigma_y \propto d^{-1/2}$ where $d$ is the mean grain diameter, we introduce:
\begin{equation}
    f_{\text{HP}} = 1 + \alpha_{\text{HP}}(1 - \tilde{c}), \qquad \alpha_{\text{HP}} = 0.06
\end{equation}
Fine-grained microstructures ($\tilde{c} < 0.5$) exhibit enhanced hardness due to grain-boundary pile-up strengthening.

\paragraph{Pore irregularity and grain count.}
\begin{align}
    f_{\text{var}}^{(HV)}  &= 1 - 2 \times 10^{-5} \cdot \bar{\sigma}^2_c \\
    f_{N_s}^{(HV)} &= 1 + 2 \times 10^{-4} (N_s - 80)
\end{align}

\section{Phase 6: Machine Learning Surrogate and Inverse Design}
\label{sec:phase6}

\subsection{Pipeline Architecture}

The ML surrogate module (\texttt{ml\_surrogate.py}) orchestrates the complete Phases~2--5 loop to construct a structure--property database, then trains independent regression models for each target property. The pipeline proceeds in four stages:

\begin{enumerate}
    \item \textbf{Database construction}: Automated generation and simulation of $n_{\text{samples}} = 200$ stochastic RVE configurations.
    \item \textbf{Feature extraction}: Transformation of microstructural descriptors into a fixed-length feature vector.
    \item \textbf{Model training}: Independent gradient boosting regressors for each of 6 target properties.
    \item \textbf{Inverse design}: Grid-search over the 2D process parameter space $(\phi, N_s)$ using database-fitted feature synthesis.
\end{enumerate}

\subsection{Database Construction}
\label{sec:database}

The database loop generates $n_{\text{samples}}$ unique microstructures by independently sampling the two primary process parameters from a pseudo-random number generator (NumPy \texttt{default\_rng}, seed = 42):
\begin{align}
    \phi_i &\sim \mathcal{U}(0.005,\; 0.035) \quad \text{(continuous uniform)} \\
    N_{s,i} &\sim \mathcal{U}\{20,\; 150\} \quad \text{(discrete uniform)} \\
    s_i &\sim \mathcal{U}\{0,\; 2^{31}-1\} \quad \text{(Voronoi seed)}
\end{align}

For each configuration $i$, the loop executes:
\begin{enumerate}
    \item \textbf{Phase 2}: Generate Voronoi RVE $\to$ compute chord-length descriptors $\mathbf{d}_i$.
    \item \textbf{Phase 3--4}: Run multi-physics homogenisation (FE or analytical fallback) $\to$ obtain $(k_{\text{eff}}, E_{\text{eff}}, \alpha_{\text{eff}})_i$.
    \item \textbf{Phase 5}: Extract density $\rho_{\text{eff},i}$, porosity $\phi_i$, and Vickers hardness $HV_i$.
    \item \textbf{Record}: Concatenate $(\mathbf{d}_i, \mathbf{y}_i)$ into a JSON-serialisable dictionary.
\end{enumerate}

The auto-detection mechanism falls back gracefully: if PyMAPDL is unavailable, the analytical models from \S\ref{sec:fallback} are invoked instead. Failed configurations (e.g., insufficient candidate voxels for the target porosity at extreme $\phi$/$N_s$ combinations) are logged and skipped. The resulting database is serialised as \texttt{simulation\_database.json}.

\subsection{Feature Engineering}
\label{sec:features}

The 9-dimensional feature vector for each RVE is derived from the microstructural descriptors computed in Phase~2:
\begin{equation}
    \mathbf{x}_i = \begin{pmatrix}
        \phi_i & \bar{c}_{x,i} & \bar{c}_{y,i} & \bar{c}_{z,i} & \sigma^2_{c,x,i} & \sigma^2_{c,y,i} & \sigma^2_{c,z,i} & \bar{c}_{\text{avg},i} & N_{s,i}
    \end{pmatrix}^\top \in \mathbb{R}^9
\end{equation}

Each feature captures a distinct physical aspect of the microstructure:

The target vector comprises six engineering properties:
\begin{equation}
    \mathbf{y}_i = \begin{pmatrix}
        k_{\text{eff}} & E_{\text{eff}} & \alpha_{\text{eff}} & \rho_{\text{eff}} & \phi_{\%} & HV
    \end{pmatrix}^\top_i \in \mathbb{R}^6
\end{equation}

No feature normalisation or standardisation is applied, as gradient boosting is invariant to monotone transformations of features (it uses rank-based splitting thresholds).

\subsection{Gradient Boosting Regression: Algorithm}
\label{sec:gbr}

For each target $y_j$, an independent Gradient Boosting Regressor (GBR) is trained following the stagewise additive framework of Friedman~\cite{friedman2001}. The algorithm constructs an ensemble of $M$ regression trees $\{h_m\}_{m=1}^M$ that sequentially correct the residual errors of the current ensemble.

\subsubsection{Loss Function}

We adopt the least-squares loss:
\begin{equation}
    L(y, F(\mathbf{x})) = \frac{1}{2}(y - F(\mathbf{x}))^2
\end{equation}
whose negative gradient (pseudo-residual) at iteration $m$ reduces to the ordinary residual:
\begin{equation}
    \tilde{r}_{i,m} = -\left[\frac{\partial L(y_i, F(\mathbf{x}_i))}{\partial F(\mathbf{x}_i)}\right]_{F = F_{m-1}} = y_i - F_{m-1}(\mathbf{x}_i)
    \label{eq:pseudoresid}
\end{equation}

\subsubsection{Boosting Iterations}

The ensemble is initialised with the empirical mean:
\begin{equation}
    F_0(\mathbf{x}) = \bar{y} = \frac{1}{n} \sum_{i=1}^n y_i
\end{equation}

At each stage $m = 1, \ldots, M$:
\begin{enumerate}
    \item Compute pseudo-residuals $\tilde{r}_{i,m}$ via Eq.~\eqref{eq:pseudoresid}.
    \item Draw a stochastic subsample of fraction $\rho = 0.8$ (without replacement) from the training set to reduce variance and correlation between successive trees.
    \item Fit a regression tree $h_m(\mathbf{x})$ of maximum depth $d_{\max} = 5$ to the pseudo-residuals on the subsample.
    \item Update the ensemble with a learning rate (shrinkage) $\eta = 0.1$:
    \begin{equation}
        F_m(\mathbf{x}) = F_{m-1}(\mathbf{x}) + \eta \cdot h_m(\mathbf{x})
    \end{equation}
\end{enumerate}

The shrinkage parameter $\eta < 1$ regularises the ensemble by requiring more trees to fit the data, effectively performing a form of $L_2$ regularisation in function space~\cite{friedman2001}. The total number of boosting stages is $M = 200$.

\subsubsection{CART Splitting Criterion}

Each tree $h_m$ is a Classification and Regression Tree (CART) that recursively partitions the feature space. At each internal node, the split $(j^*, t^*)$ on feature $j$ at threshold $t$ is chosen to minimise the weighted sum of squared deviations in the two child nodes:
\begin{equation}
    (j^*, t^*) = \arg\min_{j, t} \left[ \sum_{i \in R_L(j,t)} (\tilde{r}_i - \bar{r}_L)^2 + \sum_{i \in R_R(j,t)} (\tilde{r}_i - \bar{r}_R)^2 \right]
    \label{eq:split}
\end{equation}
where $R_L$ and $R_R$ are the left and right child regions and $\bar{r}_L$, $\bar{r}_R$ are the corresponding mean pseudo-residuals. Each leaf predicts the mean of its contained pseudo-residuals. The maximum depth $d_{\max} = 5$ allows up to $2^5 = 32$ leaf nodes per tree, permitting interaction effects between feature pairs.

\subsection{Model Training and Evaluation}
\label{sec:evaluation}

The 200-sample database is partitioned via stratified random splitting into a training set ($n_{\text{train}} = 150$, 75\%) and a held-out test set ($n_{\text{test}} = 50$, 25\%) using a fixed random seed for reproducibility. Six independent GBR models are trained---one for each target property.

\subsubsection{Evaluation Metrics}

Model quality is assessed on the test set using two complementary metrics:

\paragraph{Coefficient of determination ($R^2$).}
Measures the proportion of target variance explained by the model:
\begin{equation}
    R^2 = 1 - \frac{\text{SS}_{\text{res}}}{\text{SS}_{\text{tot}}} = 1 - \frac{\sum_{i=1}^{n_{\text{test}}}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n_{\text{test}}}(y_i - \bar{y})^2}
\end{equation}
$R^2 = 1$ indicates perfect prediction; $R^2 = 0$ corresponds to a na\"ive mean predictor.

\paragraph{Mean absolute error (MAE).}
Provides an interpretable scale-dependent error measure:
\begin{equation}
    \text{MAE} = \frac{1}{n_{\text{test}}} \sum_{i=1}^{n_{\text{test}}} |y_i - \hat{y}_i|
\end{equation}

\subsubsection{Per-Property Analysis}

\paragraph{Density and porosity ($R^2 > 0.96$).}
These properties follow near-linear rule-of-mixtures relationships with porosity; the GBR captures this trivially. Feature importance confirms $\phi$ accounts for $> 94\%$ of the predictive signal.

\paragraph{Thermal conductivity ($R^2 \approx 0.84$).}
The Maxwell--Eucken base is porosity-dominated, but the grain-boundary scattering correction introduces chord-length dependence. Feature importance reveals $\bar{c}_x$ (0.24), $\bar{c}_{\text{avg}}$ (0.19), and $\bar{c}_z$ (0.16) collectively contribute $> 50\%$ of the predictive power, confirming the microstructure-aware corrections create genuine multi-feature signal.

\paragraph{CTE ($R^2 \approx 0.83$).}
Grain count $N_s$ dominates (importance = 0.67), reflecting the grain-boundary constraint correction $f_{\text{gb}}^{(\alpha)}$ which scales linearly with $N_s$. The remaining variance is distributed across porosity and chord features.

\paragraph{Vickers hardness ($R^2 \approx 0.80$).}
The Rice--Duckworth exponential and Hall--Petch corrections create a coupled porosity--grain-size dependence. Porosity dominates (0.75), with $N_s$ contributing a secondary 0.11.

\paragraph{Young's modulus ($R^2 \approx 0.34$).}
The lowest $R^2$ reflects the aggressive pore aspect-ratio correction ($\alpha_{\text{asp}} = 0.40$), which introduces large scatter via the chord-length coefficient of variation. The $N_s$ feature importance (0.49) suggests the grain-count dependence amplifies noise. Improvement strategies include: (i) increasing the database to $n > 500$; (ii) reducing $\alpha_{\text{asp}}$ if the physical motivation is weak; or (iii) using a random forest ensemble with deeper trees.

\subsection{Feature Importance: Mean Decrease in Impurity}
\label{sec:importance}

GBR feature importance is computed via the \emph{mean decrease in impurity} (MDI) across all trees in the ensemble. For feature $j$:
\begin{equation}
    \text{MDI}(j) = \frac{1}{M} \sum_{m=1}^{M} \sum_{t \in T_m} \frac{n_t}{n_{\text{train}}} \cdot \Delta I(j, t)
\end{equation}
where $T_m$ is the set of internal nodes in tree $m$ that split on feature $j$, $n_t$ is the number of training samples reaching node $t$, and $\Delta I(j,t)$ is the reduction in squared-error impurity achieved by the split (Eq.~\ref{eq:split}). The values are normalised to sum to 1 across all features.

Key observations from the feature importance analysis:
\begin{itemize}
    \item \textbf{Porosity} dominates density (0.94), HV (0.75), and the porosity self-prediction (0.99), as expected from the physics.
    \item \textbf{Grain count ($N_s$)} is the leading predictor for $E_{\text{eff}}$ (0.49) and CTE (0.67), validating that the grain-count corrections in the analytical fallback create learnable signal.
    \item \textbf{Chord-length features} ($\bar{c}_d$, $\sigma^2_{c,d}$) contribute most to $k_{\text{eff}}$, where grain-boundary scattering and tortuosity corrections dominate the microstructural modulation.
    \item The non-zero importance of \emph{all 9 features} for most targets confirms that the microstructure-aware analytical models successfully project the property space onto the full descriptor set, avoiding the trivial one-dimensional mapping that would result from classical porosity-only formulas.
\end{itemize}

\subsection{Polynomial-Fitted Inverse Design}
\label{sec:inverse}

Given a target property value $y^*$ (e.g., $k_{\text{eff}} = 3.0$~\si{W.m^{-1}.K^{-1}}$), the inverse design module searches the 2D process parameter space $(\phi, N_s)$ to identify configurations that minimise the surrogate prediction error:
\begin{equation}
    (\phi^*, N_s^*) = \arg\min_{\phi \in [\phi_{\min}, \phi_{\max}],\; N_s \in [N_{s,\min}, N_{s,\max}]} \left| f_\theta\!\left(\mathbf{x}(\phi, N_s)\right) - y^* \right|
    \label{eq:inverse}
\end{equation}
where $f_\theta$ denotes the trained GBR surrogate.

\subsubsection{Feature Synthesis via Quadratic Regression}

The 2D process parameters $(\phi, N_s)$ do not uniquely determine the full 9D feature vector $\mathbf{x}$, since the chord-length descriptors depend on the specific stochastic Voronoi realisation. Na\"ive approximations such as $\bar{c} \approx N / N_s^{1/3}$ and $\sigma^2_c \approx 0.1 \cdot \bar{c}$ produce out-of-distribution feature vectors that lie far from the training manifold, causing the tree ensemble to extrapolate unpredictably.

We resolve this by fitting empirical polynomial maps from the training database. For each of the 7 chord features $c_k \in \{\bar{c}_x, \bar{c}_y, \bar{c}_z, \sigma^2_{c,x}, \sigma^2_{c,y}, \sigma^2_{c,z}, \bar{c}_{\text{avg}}\}$, a quadratic polynomial is fit via ordinary least squares (OLS):
\begin{equation}
    \hat{c}_k(\phi, N_s) = \beta_{k,0} + \beta_{k,1}\phi + \beta_{k,2}N_s + \beta_{k,3}\phi^2 + \beta_{k,4}N_s^2 + \beta_{k,5}\phi N_s
    \label{eq:poly_map}
\end{equation}
The design matrix for 200 training records is:
\begin{equation}
    \mathbf{Z} = \begin{pmatrix}
        1 & \phi_1 & N_{s,1} & \phi_1^2 & N_{s,1}^2 & \phi_1 N_{s,1} \\
        \vdots & \vdots & \vdots & \vdots & \vdots & \vdots \\
        1 & \phi_{200} & N_{s,200} & \phi_{200}^2 & N_{s,200}^2 & \phi_{200} N_{s,200}
    \end{pmatrix} \in \mathbb{R}^{200 \times 6}
\end{equation}
and the coefficient vector is estimated as $\hat{\bm{\beta}}_k = (\mathbf{Z}^\top \mathbf{Z})^{-1} \mathbf{Z}^\top \mathbf{c}_k$.

This approach guarantees that the synthesised features $\hat{c}_k(\phi, N_s)$ are smooth functions of the process parameters and lie near the centroid of the training distribution, since OLS projections cannot extrapolate beyond the convex hull of the design matrix.

\subsubsection{Grid Search Procedure}

The search evaluates a uniform $n_g \times n_g$ grid ($n_g = 50$, yielding 2500 candidate configurations) over:
\begin{equation}
    \phi \in [0.005, 0.035], \qquad N_s \in [20, 150]
\end{equation}

For each grid point $(\phi_j, N_{s,k})$:
\begin{enumerate}
    \item Synthesise chord features via Eq.~\eqref{eq:poly_map}: $\hat{\mathbf{c}}_{jk} = (\hat{c}_1(\phi_j, N_{s,k}), \ldots, \hat{c}_7(\phi_j, N_{s,k}))$.
    \item Assemble the full feature vector: $\mathbf{x}_{jk} = (\phi_j,\; \hat{\mathbf{c}}_{jk},\; N_{s,k})^\top$.
    \item Query the surrogate: $\hat{y}_{jk} = f_\theta(\mathbf{x}_{jk})$.
    \item Compute error: $\varepsilon_{jk} = |\hat{y}_{jk} - y^*|$.
\end{enumerate}

The global minimum $(\phi^*, N_s^*) = \arg\min_{j,k}\; \varepsilon_{jk}$ is reported with its predicted value $\hat{y}^*$ and absolute error $\varepsilon^*$. The full $n_g \times n_g$ prediction grid is visualised as a 2D filled contour map (magma colourmap) with the target iso-line $\hat{y} = y^*$ overlaid as a dashed contour and the optimal point annotated.

\subsection{Model Persistence}

Trained GBR models are serialised to disk via \texttt{joblib} for deployment without retraining:
\begin{lstlisting}[language=Python, caption=Model serialisation and loading.]
from joblib import dump, load

# Save all models
for target, model in models.items():
    dump(model, f"models/{target}.joblib")

# Load for inference

k_pred = model.predict(new_features)
\end{lstlisting}

This enables rapid inference ($< 1$~ms per prediction) for integration into design optimisation loops or interactive tools without re-executing the training pipeline.

\section{Conclusion}
\label{sec:conclusion}

This work presents an integrated computational materials design pipeline for porous Wollastonite ceramics that operates in both licensed FE and license-free analytical modes without workflow modification. The key methodological contributions are:

\begin{enumerate}
    \item \textbf{Microstructure-aware analytical bounds}: Classical homogenisation formulas (Maxwell--Eucken, Mori--Tanaka, Turner ROM) are augmented with five categories of physically motivated correction factors --- grain-boundary scattering, tortuosity, pore irregularity, grain-size effects, and grain-count dependence --- parameterised by chord-length descriptors computed directly from the RVE voxel array.

    \item \textbf{Controlled stochastic perturbation}: Hash-seeded Gaussian noise ($\text{CV} = 1.5$--$2.0\%$) and reference temperature sampling ($T_{\text{ref}} \pm 30$~\si{\celsius}) break the deterministic porosity-only mapping and provide the variance necessary for meaningful ML training.

    \item \textbf{Database-fitted inverse design}: Polynomial regression maps from the training database generate in-distribution chord features for the surrogate query grid, eliminating artefacts from na\"ive feature approximation.
\end{enumerate}

The resulting gradient boosting surrogates achieve $R^2 \in [0.32, 0.95]$ across the six target properties, with the variation reflecting genuine differences in microstructural sensitivity. The framework enables rapid virtual prototyping and target-driven microstructure selection for ceramic engineering applications.
